Colophon¶

The documentation sites are generated using MkDocs and use the Kilsbergen theme. I post-process the MkDocs output to minify the HTML using Minimizer in a Git post-receive hook, and then I serve everything with Miniserver. This index page itself uses a custom variation of the Kilsbergen stylesheet.
